# 要件定義

### プロジェクトの目的
- ECサイトの構築
- ユーザーフレンドリーな購入フロー
- モバイル対応

### 機能要件
- カート機能
- 支払い方法（クレジットカード、銀行振込）
- ユーザー登録（メールアドレス、パスワード）
- 商品一覧表示（トップページ）
- レスポンシブデザイン（スマホ対応）
- 商品検索機能（追記）
- 商品レビュー機能（追記）
- 会員ポイントシステム（追記）

### 非機能要件
- ページ読み込み時間2秒以内
- SSL対応
- データベース（MySQL）
- セキュリティ対策優先
